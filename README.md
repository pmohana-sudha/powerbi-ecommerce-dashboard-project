# Madhav E-Commerce Sales Dashboard

This project demonstrates an interactive sales dashboard built in Power BI to analyze and visualize e-commerce sales data. It uses two CSV tables: **Orders** and **Details**.

## Features

- **Cards**: Display key metrics including total Amount, Profit, Quantity, and Average Order Value (AOV).  
- **Stacked Column Chart**: Visualizes Profit by Month to track trends over time.  
- **Stacked Bar Chart**: Shows Profit by Sub-Category for detailed product insights.  
- **Donut Charts**:  
  - Quantity by Category  
  - Quantity by Payment Mode  
- **Stacked Column Chart**: Amount by Customer Name to identify top customers.  
- **Stacked Bar Chart**: Amount by State to analyze regional performance.  
- **Filters / Slicers**:  
  - Quarter filter for dynamic time-based analysis  
  - State drop-down filter to view data for a specific state across all charts  

The dashboard allows users to explore sales data dynamically and gain insights into revenue, profit, product performance, and customer trends.  

## Tools Used

- **Power BI Desktop**  
- **CSV datasets** (Orders & Details)  

## Purpose

Created as a learning project to showcase skills in **data visualization, interactive dashboards, and analytical thinking**.
